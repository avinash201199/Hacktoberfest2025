What is Hacktoberfest?

Hacktoberfest is a month-long open-source celebration organized every October by DigitalOcean (with community sponsors). It encourages developers, designers, writers, and everyone in tech to contribute to open-source projects.

How to Participate - Step by Step

Follow these simple steps to participate in Hacktoberfest 2025 and make your first open source contribution!

1

Register / Sign Up

  • Visit Hacktoberfest.com
  • Sign in with your GitHub or GitLab account
  • Start earning your digital badge
Registration opens on October 1st, 2025
2

Find Suitable Projects

  • Only repositories with the hacktoberfest topic are eligible
  • Look for issues labeled good first issue or help wanted
  • You can contribute to non-code areas: docs, translation, design, tutorials
  • Check Hacktoberfest's official event listings and community project lists
Browse Projects Good First Issues
3

Make Pull Requests

  • Fork the repository you want to contribute to
  • Branch - Create a new branch for your changes
  • Commit - Make your changes with clear commit messages
  • PR - Submit a pull request to the original repository
  • Follow the repo's CONTRIBUTING.md guidelines
  • Use clear PR titles and descriptions
# Quick workflow
git clone https://github.com/YOUR_USERNAME/REPO.git
git checkout -b feature/your-feature
git add .
git commit -m "Add: your changes"
git push origin feature/your-feature
4

Get PRs Accepted

Repository must have hacktoberfest topic

PR should be merged or labeled hacktoberfest-accepted

Aim for 6 accepted PRs in 2025

Engage with maintainers - respond to feedback and revise if needed!

Important Tips

DO

  • Focus on quality, not quantity
  • Read repository guidelines
  • Open issues before making PRs
  • Write clear commit messages

DON'T

  • Submit meaningless PRs (typos only)
  • Contribute to ineligible repos
  • Ignore maintainer feedback
  • Spam with low-quality PRs
